## About

The legendary first-person shooter that defined a genre.

## Features

- Classic gameplay
- All original levels
- Keyboard/mouse controls
- Save games

## Installation

### From App Store
Open the **App Store** in zOS and search for "DOOM".

### Via npm
```bash
npm install @zos-apps/doom
```

## Usage

- WASD to move
- Mouse to aim
- Click to shoot

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `WASD` | Move |
| `Space` | Use/Open |
| `1-7` | Weapons |
| `Esc` | Menu |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
